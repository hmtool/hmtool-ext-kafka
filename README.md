# kafka-spring

#### 项目介绍
kafka基础spring版
基于spring,apache的kafka-clients的包装
支持jdk11、springboot1 springboot2、springcloud1,springcloud2 使用简单快捷！
公司生产上使用、此为抽离出的简要版

-----------------------------------------
0.0.1         目前版本  

####功能说明
0.0.1版更新功能
    1.实现动态更新生产配置、
    2.动态关闭消费者配置
    3.分片规则默认随机
    4.动态根据所配置的信息创建对应的spring bean、消费
